## 🚀 The Mission
**Reelist** is designed for the modern cinephile who wants more than just a list. We believe that tracking your cinematic journey should be as rewarding as the films themselves. By bridging the gap between discovery and reflection, Reelist helps you curate your watches, master your queue, and celebrate every milestone along the way.

### ✨ Why Reelist?
*   **Intuitive Tracking:** Go beyond "watched" and "unwatched"—track specific episodes, jot down notes, and rate titles with precision.
*   **Personalized Discovery:** Our trending and popular feeds ensure you’re always one step ahead of the conversation.
*   **Gamified Milestones:** Earn thematic badges that evolve with your viewing habits, turning your stats into a collection of achievements.
*   **Adaptive Atmosphere:** Switch between curated themes like *Warm Cream* or *Deep Forest* to match your mood.
*   **Data-Driven Insights:** Understand your viewing patterns with beautiful, interactive visualizations.

## 🛠️ Built With
*   **Frontend:** React / Vite
*   **Styling:** Custom CSS with dynamic theme variables
*   **Intelligence:** The Movie Database (TMDB) API
*   **Visualization:** Recharts
*   **Icons:** Lucide React

## 📦 Local Setup

1. **Clone the repository:**
   
```bash
git clone [https://github.com/yourusername/reelist.git](https://github.com/yourusername/reelist.git)
cd reelist
```

2. **Install dependencies:**

```bash
npm install
```

3. **Configure your environment:**
Create a .env file in the root directory and add your API key:

```bash
VITE_TMDB_API_KEY=your_api_key_here
```
4. **Launch the experience:**

```bash
npm run dev
```
